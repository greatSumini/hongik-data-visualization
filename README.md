# 일본군 성노예제 진실의 역사 : 소녀, 나비, 빛

홍익대학교 디자인학부 시각디자인학과 기업적디자인스튜디오 과목의 제출물입니다.

Next.js를 사용해 구현했습니다.

### Getting Started

```bash
npm install
npm run dev

Visit http://localhost:3000
```
